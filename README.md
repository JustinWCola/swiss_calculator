# swiss_calculator

一个用于瑞士制赛事的对手分（Opponent Points）与轮次配对计算器，适合小型比赛的赛程生成、比分填写与结果导入/导出。

**主要功能**
- **自动配对**：后续轮次可在“顺位配对 / 首尾配对 / 分组配对”三种模式间切换。
- **比分与评价输入**：可在每轮为每场填写比分与评价分。
- **导出/导入**：将当前赛事设置与已填数据导出为 JSON，随后可导入恢复。
- **本地持久化**：页面会将设置自动保存到浏览器本地存储。

**快速开始**
- 打开项目中的 [index.html](index.html)（双击或在浏览器中打开）。
- 在某些浏览器或平台上直接打开本地文件可能受限，建议使用静态文件服务器：

	- 使用 Python (在项目根目录运行)：

		```bash
		python -m http.server 8000
		```

		然后访问 http://localhost:8000/index.html。

	- 或使用 Node.js 工具，例如 `npx serve`：

		```bash
		npx serve .
		```

**使用说明**
- 在页面中设置参赛队伍数量、总轮次数、晋级/淘汰线，点击 “生成赛程”。
- 第一轮也可以在列头切换首轮配对方式。
- 在每场比赛中填写比分和评价分。
- 后续轮次会根据每轮列头选择的配对方式自动生成配对。
- **轮次切换**：每一轮都可以在列头切换“顺位配对 / 首尾配对 / 分组配对”。
- 点击“导出数据”下载 JSON 文件；使用“导入数据”可恢复之前导出的状态。

**配对方式说明**
- **顺位配对**：在组内按排名顺序配对。（1-2，3-4）
- **首尾配对**：在组内将首位与末位配对、次首与次末配对。（1-4，2-3）
- **分组配对**：在组内按排名再分成两小组，小组内排名相同的配对。（1-3，2-4）
