# swiss_calculator

Swiss System Opponent Points and Round Calculator

## Usage

1. Open `/tmp/workspace/JustinWCola/swiss_calculator/index.html` in a browser.
2. Configure team count, format (BO1/BO3/BO5), rounds, and qualified teams.
3. Provide custom team names (one per line) and click `生成赛程`.
4. Drag teams in the first-round seeding area to change round-1 matchups.
5. Adjust each match win rate slider to simulate results.
6. View Swiss standings, opponent points, and next-round pairing probability cards.
