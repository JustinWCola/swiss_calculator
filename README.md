# swiss_calculator

Swiss System Opponent Points and Round Calculator

## Usage

1. Open `/tmp/workspace/JustinWCola/swiss_calculator/index.html` in a browser.
2. Configure team count and rounds, then click `生成赛程`.
3. In round 1 column, directly select the first-round matchup.
4. Fill scores and evaluation points for each team in each match.
5. Round 2 and later matchups are auto-generated by: opponent points (desc) then evaluation points (desc), paired as 1v2, 3v4, ...
6. Click `导出数据` to download current settings + bracket inputs as JSON.
7. Click `导入数据` to restore a previously exported JSON file.
